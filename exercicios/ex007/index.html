Princípais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag < b >. (não semântica)

Nesta frase, temos um termo em destaque utilizando a tag < strong >. (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico utilizando a tag < i >. (não semântica)

Nesta frase, temos um termo em ênfase utilizando a tag < em >. (semântica)

Texto Marcado

podemos também criar um texto marcado utilizando a tag < mark >.

e no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande com a tag < big > e um texto pequeno com a tag < small > neste parágrafo.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado com a tag < del >.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag < ins >. (semântica)

Existe também o sublinhado com a tag < u >. (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, utilizamos a tag < sup >.

Texto subscrito

Para inserir coisas do tipo H20, utilizando a tag < sub >.
